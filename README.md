1. Project Objective
Transform raw, multi-dimensional sales data from an Indian e-commerce business into an interactive, real-time dashboard to drive data-driven decision-making. 

2. Key Performance Indicators (KPIs)
Total Revenue & Profit: Overall financial health of the business.
Sales by Region: Performance across different Indian states and zones.
Category-Wise Analysis: Identifying top-selling and most profitable product lines.
Order Trends: Monthly and quarterly growth patterns. 

3. Data Cleaning & Preparation
Ensured data consistency (e.g., date formats, currency, and data types).
Handled missing values and anomalies in the raw dataset.
Created helper columns for regional grouping and fiscal year analysis. 

4. Dashboard Features
Slicer Interactivity: Dynamic filtering by Year, Month, Region, and Product Category.
Visualizations: Includes Column Charts for trends, Tree Maps for category distribution, and Heat Maps for geographic performance.
User Protection: Locked dashboard layout while keeping slicers interactive to prevent accidental edits.
